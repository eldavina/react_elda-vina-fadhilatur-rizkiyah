## Kesalahan Umum Membuat Resume

- Salah ketik (typo)
- Tidak berstruktur
- Sangat klise dan banyak buzzwords
- Tidak ada achievements
- Terlalu panjang
- CV umum atau tidak spesifik

## Konten dalam CV

- Relevan
- Ada angka lebih baik
- Hindari Buzzwords
- Singkat
- Perhatikan keyword
- Pakai font yang tepat
- Baca ulang
